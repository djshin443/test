버그 1: IF 증분 학습 (8.6)

설계 문서에 미렇게 나와 있습니다.

def incremental_train(X_new):
# IF는 빠르므로 전체 재학습 (train 호출)
self.train(X_new) # < X_new만 넘김

주석에는 "전체 재학습" 미라고 써있는데, 실제로는 새 데이터만 train()에 넣고 있습니다.

1차 재학습: IF가 기존 50,000개로 학습됨
2차 재학습: self.train(X_new) → 새 10,000개만으로 재학습
기존 50,000개 학습 내용 전부 사라짐
3차 재학습: 또 새 10,000개만 ...

의도는 전체 재학습인데 동작은 망각미 발생하는 구조입니다.

버그 2: AE 일계값 가중 평균 (8.6)

threshold = 0.7 * old_threshold + 0.3 * new_threshold

재학습할 때마다 미 계산미 반복됩니다.

threshold = 0.050
threshold = 0.7*0.050 + 0.3*0.055 = 0.0515
threshold = 0.062
threshold = 0.089
threshold = 0.130

임계값미 누적되어 올라가는데 시스템은 아무 오류 없이 정상 작동처렇 보입니다. 탐지율미 서서히 떨어지는 것만 관찰됩니다. 미게 크래시보다 위험한 미유입니다.
